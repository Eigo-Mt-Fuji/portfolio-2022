# おしごとをドラクエのじゅもんに例えてみた

---
# はじめに

- 開発の業務にも、わかりやすいものと、そうでないものが存在する
- どんなお仕事をしているのか、色んな人にわかってもらえるようにしたい
- 今回は、ドラクエに出てくるじゅもんに例えて、おしごとをざっくり説明したいと思います

---
# やりかた

- 実際にやっていた業務で、傍(はた)から見るとなんだかわかりにくいものを８つピックアップ
- 2021年に実際に取り組んでいたAWSを使ったシステムかいぜん提案から、抜粋しました。
- ８つの業務をじゅもんで置き換えていきます。

---
# 本題

- インパス ... 調査・ヒアリングを通じてシステム運用・組織の事情を視える化する。実世界ではMP消費かなり多め。インパス使わない派もいるので、たまにこっそり使う羽目に..
- キアリー ... 毒におかされたサーバ(脆弱性、メモリリークなど)を刷新
- キアリク ... マヒ状態のシステム(担当者不在、ソース不備でデプロイ不能)を刷新
- ホイミ　 ... ほんのすこしプログラム修正するなどしてシステムを延命させる
- スクルト ... 全体的に少しずつシステム監視・運用を強化する 大事な呪文
- フバーハ ... 全体的に少しだけプロダクト運用を強化する 大事な呪文
- ベホマラー ... 全体的にシステムを延命させる (こまめにホイミかけたらいいんじゃ..?
- ザオラル ... 死んだシステムを復活させる (こまめにホイミかけたらいいんじゃ..?

---

# 解説

- どんなことが起こるか?
    - 回復呪文はタイミングが大切
    - MPが足りないこともある
    - じゅもんが効かない敵と遭遇
    - ダメージに回復が追いつかない

---

# 解説

- ドラクエの世界でのその呪文の効果 と 業務における施策の効果はにてるか

---

# まとめると

- 以上、おしごとをドラクエのじゅもんに例える話でした。
    - 頑張りすぎてMPが切れないように注意してお仕事しましょう。
    - 目立たないけど大事な改善(補助)系の役割を忘れないでください
    - さくせんとパーティも大事です。
